**===Personal Profile Website===**



**===About The Project===**

This project is a dynamic, multi-page personal profile developed for Midterm Activity 1. 

It showcases my skills, projects, and certifications while demonstrating proficiency in 

front-end web development, version control, and cloud-based tool integration. 



The original static single-page application (SPA) was refactored into a Multi-Page Application (MPA) 

using a dedicated Git branch to fulfill the dynamic functionality and software design requirements 

of the midterm rubric.



**===Tech Stack \& Tools===**

**HTML5 \& CSS3:** Semantic markup and modern custom dark-theme UI layout.

**JavaScript (ES6):** Interactivity, Scroll-Reveal (Intersection Observer API), and dynamic async database manipulation (`fetch`).

**MongoDB Atlas NoSQL Database:** Connected via the cloud Data API to directly record, store, and manage user contact inquiries.

**Git \& GitHub:** Version control featuring a multi-page file refactoring branch strategy.
